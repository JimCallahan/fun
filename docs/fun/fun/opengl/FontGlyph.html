FontGlyph

Related Docs: object FontGlyph | package opengl

case class FontGlyph(index: Int, char: Char, advance: Double, bbox: BBox2d) extends Ordered[FontGlyph] with Product with Serializable

Metric information about a specific textured glyph.

index

The index (depth) of the glyph within the 3D texture.

char

The character represented by the glyph.

advance

The amount to advance the position (in texture coords) after rendering the glyph.

bbox

The bounds (relative to the glyph anchor in texture coords) of the visible glyph.
