case class KeyboardEvent(nanos: Long, character: Char, keyCode: Int, action: KeyAction, shift: ShiftKeyState, control: ControlKeyState, meta: MetaKeyState) extends Product with Serializable

An event involving a change in keyboard state.

nanos

The time in nanoseconds of the current event.

character

The character involved.

keyCode

The org.lwjgl.input.Keyboard._ code for the key involved.

action

The change of key state that caused the event.

shift

The state of the SHIFT key during the event.

control

The state of the CONTROL key during the event.

meta

The state of the META key during the event.
